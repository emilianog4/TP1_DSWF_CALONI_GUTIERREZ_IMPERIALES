Bitácora de Desarrollo - TP1

1. Decisiones de Diseño

El equipo optó por una estética "Tech-Professional". Seleccionamos una paleta de colores basada en tonos azul pizarra (`#1E293B`) y azul brillante (`#3B82F6`) sobre fondos claros para garantizar la legibilidad y transmitir profesionalismo. Utilizamos Google Fonts (Montserrat para títulos e Inter para el cuerpo) para mejorar la experiencia visual.

2. Dificultades y Resoluciones

  • Desafío: Implementar el diseño adaptable (Responsive Design) para los tres breakpoints solicitados (400px, 900px, 1200px).
    Solución: Se utilizaron Media Queries en CSS para reorganizar los elementos de la tarjeta de perfil, pasando de un diseño vertical en móviles a uno más espacioso en escritorio[cite: 1].
  • Desafío: Errores en la vinculación de funciones de JavaScript con los elementos del DOM.
    Solución: Se estandarizaron los IDs en el HTML y se utilizó `DOMContentLoaded` para asegurar que el script se ejecute solo cuando el documento esté totalmente cargado.

3. Cambios Importantes

Originalmente, se consideró usar una sola página (Single Page Application), pero se decidió seguir la consigna de archivos HTML independientes para el índice y cada integrante, facilitando así el mantenimiento y la organización del repositorio.
