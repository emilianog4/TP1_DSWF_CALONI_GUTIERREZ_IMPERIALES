**Link al proyecto desplegado:** https://tp1dswf.vercel.app/

## Descripción del Proyecto

Este trabajo práctico consiste en el desarrollo de un sitio web estático para el **IFTS N.°29**. El objetivo es demostrar habilidades en la organización de archivos, diseño responsive mediante Media Queries y la implementación de interactividad básica con JavaScript. El sitio incluye una portada grupal, páginas individuales para cada integrante y una bitácora de desarrollo detallada.

## Tecnologías Utilizadas

- **HTML5:** Estructura semántica del sitio.
- **CSS3:** Estilos prolijos y Responsive Design con breakpoints en 400px, 900px y 1200px.
- **JavaScript (Vanilla):** Funciones dinámicas para mejorar la experiencia de usuario.
- **Google Fonts:** Tipografías Montserrat e Inter.
- **Vercel:** Hosting y despliegue del proyecto.

## Estructura de Archivos

El proyecto sigue las buenas prácticas de organización de archivos solicitadas:

- `/` (Raíz): Contiene los archivos `.html` (index, bitacora y perfiles) y el `README.md`.
- `/css`: Contiene el archivo `styles.css` con la guía de estilos unificada.
- `/js`: Contiene los scripts de interactividad `script.js` y `script-perfil.js`.
- `/img`: Almacena los avatares generados por IA y recursos visuales.

## Guía de Estilos

- **Paleta de Colores:**
  - **Fondo:** `#F8FAFC`
  - **Texto Principal:** `#1E293B`
  - **Accento:** `#3B82F6`
- **Tipografías:**
  - **Títulos:** [Montserrat](https://fonts.google.com/specimen/Montserrat)
  - **Cuerpo:** [Inter](https://fonts.google.com/specimen/Inter)
- **Iconografía:** Se optó por un estilo minimalista y el uso de avatares pixel-art generados por IA para garantizar la privacidad de los integrantes.

## JavaScript e Interactividad

- **Portada (index.html):** Se implementó una función que detecta la hora del sistema para mostrar un saludo de bienvenida dinámico.
- **Páginas Individuales:** Cada tarjeta de presentación cuenta con un botón que permite cambiar el color de fondo de la tarjeta de forma aleatoria, demostrando manipulación del DOM.

## Uso de IA

- **Herramientas:** Gemini 3 Flash (Google).
- **Contenido y Código:** Se utilizó la IA como asistente técnico para la optimización de Media Queries en CSS, la estructuración de la Bitácora y la depuración de la lógica de JavaScript.
- **Imágenes:** El avatar personal fue generado y editado con el modelo de imagen de Gemini, aplicando un estilo pixel-art retro y criterios específicos de rasgos físicos para cumplir con los requerimientos estéticos y de privacidad.
